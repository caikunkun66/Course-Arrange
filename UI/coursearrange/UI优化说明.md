# 课程排课系统 UI 优化说明

## 优化概览

本次优化采用简约、现代的设计风格，全面升级了系统的视觉体验和用户交互。

## 设计系统

### 配色方案

- **主色调**: 蓝紫渐变 (#667eea → #764ba2)
- **辅助色**:
  - 成功: #10b981 (绿色)
  - 警告: #f59e0b (橙色)
  - 危险: #ef4444 (红色)
  - 信息: #3b82f6 (蓝色)

- **中性色**:
  - 文本主色: #1f2937
  - 文本次要: #6b7280
  - 边框: #e5e7eb
  - 背景: #f9fafb

### 圆角系统

- 小圆角: 4px - 用于小元素
- 中圆角: 8px - 用于按钮、输入框
- 大圆角: 12px - 用于卡片、表格
- 超大圆角: 16px - 用于对话框

### 阴影系统

- 小阴影: 用于轻微的层次感
- 中阴影: 用于卡片和表格
- 大阴影: 用于对话框和弹窗
- 超大阴影: 用于重要的模态框

## 使用建议

### 应用通用样式
在组件中添加类名即可：
```vue
<div class="modern-card">
  <div class="modern-card-title">标题</div>
  内容
</div>
```

### 自定义颜色
使用 CSS 变量：
```css
.custom-element {
  background: var(--primary-gradient);
  color: var(--text-primary);
  border-radius: var(--radius-lg);
}
```
